## Ce que propose Swiss InterWeb, concrètement

Swiss InterWeb fait de l'hébergement mutualisé sous cPanel (Linux), avec cinq formules allant du petit site perso jusqu'au WordPress professionnel bien musclé. Tous les hébergements tournent sur CloudLinux, ce qui signifie que les ressources sont isolées entre clients — autrement dit, le voisin qui fait tourner un script mal optimisé ne viendra pas ralentir votre site.

Les serveurs sont répartis entre deux datacenters suisses : l'un à Genève (Gigaplex), l'autre à La Chaux-de-Fonds (HighDC, le seul datacenter réellement certifié écologique de Suisse romande). Vos données ne quittent pas la Confédération.

Le support fonctionne en français et en anglais, du lundi au vendredi de 9h à 17h. Pas du 24/7, mais pour un hébergement mutualisé classique, c'est généralement suffisant.

**PRO WP** — la formule complète pour les sites WordPress à fort trafic ou les agences qui gèrent plusieurs sites WordPress depuis un seul hébergement.
